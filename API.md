# Frontend - Backend data API
## Report Page
### Request Data
- 기간
- 사용자 정보

### Response Data
사용자가 "장바구니"에 추가해놓은 항목 중에 지정한 기간에 생산된 정보로 응답

만약 해당하는 정보가 없으면 없다는 메세지를 명시

- (요청 정보 그대로) 기간
- 한 문장 제목
- 3~4문장 요약 단락
- Tags: 악의성, 클릭베이트
- 출처명
- 출처 URL
- **날짜 및 시간** - 크롤링된 일시? 정보가 만들어진 일시? 후자가 나을 텐데 그걸 어떻게 구하지
- 썸네일 이미지
